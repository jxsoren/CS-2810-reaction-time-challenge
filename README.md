# Reaction Time Challenge

A reaction time testing device

## Team Members
- Andrew Clark
- Josh Sorensen  
- Kevin Heivilin

## Description
Test your reaction time by touching illuminated sensors as quickly as possible. Race against a 60-second timer to achieve the highest score and climb the leaderboard.

## Hardware
- Raspberry Pi 4
- Arduino Uno
- 5 LEDs
- 5 Touch Sensors
- Display Screen (HDMI)
- Wooden Board (5-8 feet)

## How It Works
1. Touch the center sensor to start
2. A random light turns on
3. Touch the corresponding sensor to earn a point
4. Next light appears - keep going!
5. Game ends after 60 seconds
6. View your score and leaderboard
